# MTTD-R with Dynatrace DQL



### MTTD-R in a Demo Environment Built with Dynatrace DQL
MTTD-R (Mean Time to Detect and Repair / Recovery) is a metric that measures the average time required to detect an incident and return the service to normal operation. This metric is important because it reflects how quickly monitoring and operational processes can identify issues and restore service, directly impacting system reliability, user experience, and operational efficiency.

### Feature
- Filtering by Application Group or Management Zone to obtain specific application performance

- Showing trends for the last 7 days or a dynamic timeframe

- Showing applications or services that are frequently impacted

- Showing which applications or services are the root cause of problems
